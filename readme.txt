A basic, clean and minimalistic Wordpress admin theme.

== Description ==

This is a simple plugin, transforming your Wordpress backend into a more calm, clean and overall prettier place to work.

== Installation ==

This plugin doesn't need much installation or configuration.

1. Upload the 'tamed-admin-theme' directory to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==

= Does this plugin in any way influence my frontend? =

Nope, it does not. It is just a way to give a nice touch to the backend.
